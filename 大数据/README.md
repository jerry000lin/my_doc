# 大数据

## 定位

这个目录用于整理 Spark、PySpark、数据访问、数据排查和数据平台相关实践。文档和脚本分层维护，避免执行工具和复习材料混放。

## 目录说明

| 目录 | 内容边界 |
| --- | --- |
| `spark/` | Spark 和 PySpark 的机制、常用操作、性能问题和排查方法。 |
| `scripts/` | 数据库表扫描、权限检查、结果查看等可执行脚本。 |

## 维护规则

- 机制分析和操作说明写成 Markdown 文档。
- 可执行脚本放入 `scripts/`，文件名避免空格。
- 如果脚本会生成结果文件，应优先写入临时目录或被 `.gitignore` 忽略的路径。
- 文档中引用脚本时，说明输入、输出、依赖环境和失败场景。
