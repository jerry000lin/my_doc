# 前端

## 定位

这个目录用于整理前端技术栈的机制、实践边界和示例代码，重点关注 React、TypeScript、构建工具、校验模型和跨框架迁移理解。

## 目录说明

| 目录 | 内容边界 |
| --- | --- |
| `react/` | React 状态管理、Hook、组件设计、框架机制和迁移对比。 |
| `typescript/` | 类型系统、类型工具、类型建模和示例代码。 |
| `vite/` | 构建机制、开发服务器、打包边界和工程化分析。 |
| `babel/` | 编译转换、插件机制和语法降级。 |
| `zod/` | 数据校验、运行时类型约束和表单/接口边界。 |
| `vue/` | Vue 机制、组合式 API、响应式系统和迁移对比。 |
| `练手项目/` | 完整小项目、项目复盘和工程化实践，不放一次性零散片段。 |

## 维护规则

- 理论文档放在具体技术目录下。
- 示例代码放入当前技术目录的 `examples/`。
- 框架对比文档放在主要分析对象目录下，例如 Vue 到 React 的迁移理解放在 `react/`。
- 工程化内容按工具归类，避免混入框架目录。
